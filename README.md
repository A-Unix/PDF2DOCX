# PDF2DOCX
A simple python script to extract data from a pdf file and save it in a MS Word file.

## How to use

### Clone the repo
``` git clone https://github.com/Anonymous-Hash30/PDF2DOCX.git ```

### Change the PWD
``` cd PDF2DOCX ```

### Change the permissions
``` chmod +x main.py ```

### Run script
``` ./main.py ```

# Enjoy :)
